# E-Commerce Ad & Sales Dashboard
📊 Built in Power BI | Ad Spend, ROAS, Sales Tracking

## Overview
Multi-brand, multi-marketplace dashboard tracking sales performance 
and ad efficiency across Amazon, Flipkart, Myntra, Nykaa & Snapdeal.

## Key Metrics
- Actual Sales: 8M+ | Ad Spend: 411K+ | ROAS: 20.31
- Achievement Rate: 95.09%
- Brands: NovaWear, UrbanThread, StyleNest, EthniCraft, LuxeLane

## Features
- ROAS Trend by Month
- Sales vs Target variance tracking
- Ad Spend vs Revenue correlation
- Brand-wise tabbed views with marketplace filters
